## `08 // IMPLEMENTATION PIPELINE`

### 1. Create the Google Sheets environment

Open a new Google spreadsheet:

[Launch Google Sheets](https://sheets.new)

### 2. Open Apps Script

From the spreadsheet menu, navigate to:

```text
Extensions → Apps Script
```

### 3. Install the automation engine

Delete the default function from the script editor and paste the contents of:

```text
AutomationEngine.js
```

### 4. Save the project

Give the Apps Script project a recognisable name, such as:

```text
Google Sheets Automation Engine
```

### 5. Execute the pipeline

Select the following function from the Apps Script execution menu:

```javascript
runAutomationPipeline
```

Then press:

```text
Run
```

### 6. Approve permissions

The first execution may request permission to access the active Google spreadsheet.

Review the requested permissions and approve access for the script.

### 7. Verify the report

Return to the spreadsheet and confirm that the automation generated:

- Dashboard headers
- Structured transaction records
- System status values
- Processing timestamps
- Execution confirmation logs

## `09 // CORE ENGINE EXAMPLE`

```javascript
function runAutomationPipeline() {
  const spreadsheet =
    SpreadsheetApp.getActiveSpreadsheet();

  const sheet =
    spreadsheet.getActiveSheet() ||
    spreadsheet.insertSheet("Automated_Report");

  const payload = [
    {
      id: "TR-101",
      target: "Advanced AI Overview Audit",
      status: "Active"
    },
    {
      id: "TR-102",
      target: "Link-Building Supply Stream",
      status: "Optimized"
    },
    {
      id: "TR-103",
      target: "Programmatic Keyword Classifier",
      status: "Completed"
    }
  ];

  const timestamp = new Date();

  const rows = payload.map(function(item) {
    return [
      item.id,
      item.target,
      item.status,
      timestamp
    ];
  });

  sheet
    .getRange(
      sheet.getLastRow() + 1,
      1,
      rows.length,
      rows[0].length
    )
    .setValues(rows);

  Logger.log(
    "PIPELINE COMPLETE // " +
    rows.length +
    " records written"
  );
}
```

## `11 // API CONNECTION PATTERN`

Replace the simulated payload with an external API request:

```javascript
function fetchApiPayload() {
  const endpoint =
    "https://api.example.com/data";

  const options = {
    method: "get",
    contentType: "application/json",
    muteHttpExceptions: true
  };

  const response =
    UrlFetchApp.fetch(
      endpoint,
      options
    );

  const responseCode =
    response.getResponseCode();

  if (
    responseCode < 200 ||
    responseCode >= 300
  ) {
    throw new Error(
      "API request failed with status " +
      responseCode
    );
  }

  return JSON.parse(
    response.getContentText()
  );
}
```

For sensitive credentials, store tokens in Apps Script Properties rather than directly inside the source code:

```javascript
const apiToken =
  PropertiesService
    .getScriptProperties()
    .getProperty("API_TOKEN");
```
