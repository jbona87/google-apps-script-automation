## 🚀 IMPLEMENTATION PIPELINE

To deploy this engine directly to your Google Workspace environment:

1. Create a blank spreadsheet at [sheets.new](https://sheets.new).
2. Navigate to the top menu navbar and select **Extensions** > **Apps Script**.
3. Delete any default code block inside the editor and paste the entire payload from `AutomationEngine.js`.
4. Click the **Save** disk icon, select the `runAutomationPipeline` function from the dropdown menu, and hit **Run**.
